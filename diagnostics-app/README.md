## OAuth Mercado Livre (Setup rápido)

Crie um arquivo `.env.local` na raiz com:

```
MELI_CLIENT_ID=seu_client_id
MELI_CLIENT_SECRET=seu_client_secret
MELI_REDIRECT_URI=https://SEU-APP.vercel.app/api/oauth/callback
```

Depois rode `npm run dev` e, na página inicial, clique em "Conectar Mercado Livre".

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `src/app/page.tsx`. The page auto-updates as you edit the file.
